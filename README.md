# Debugger

A debugger for written in c++ working for linux.

# Build
```
git clone https://github.com/rytterm/debugger.git ~/mdbg
cd ~/mdbg/
make
```
# Running

```
cd ~/mdbg/
./mdbg [target]
```
# Utility

Uitlity inside the program will be steadily updated.

Currently avaliable:

continue/cont/c - Continue execution of program.
